# JointResearch
PWRI’s joint research with other organization related OPERA

国立研究開発法人土木研究所は『自律施工技術基盤OPERAを活用した機械土工の生産性向上に関する共同研究』を公募し
応募のあった10団体と2022年9月より共同研究を開始しました。
本共同研究は2024年度末までの計画として推進します。
